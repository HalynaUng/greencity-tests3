# Test Case 3

**Title:** Checking the addition of a habit in your own account  
**Priority:** Medium  
**Precondition:** The user is registered on the site and has an active link to the site.

| Step # | Step Action                                  | Data                                                        | Expected Result                                                                                                   |
|--------|---------------------------------------------|------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| 1      | Follow the link                             | https://www.greencity.cx.ua/#/greenCity/about             | The page is opened                                                                                                |
| 2      | Click the My Account tab                     |                                                            | The user profile is opened                                                                                        |
| 3      | Select the New Habit button                  |                                                            | A menu with habits that can be selected is opened                                                               |
| 4      | Click the button -- Do not use disposable packages |                                                        | This habit is added to the user profile                                                                          |
| 5      | Look at the user status                      |                                                            | In the profile, in the bottom-left corner, the numbering of habits in progress is shown (in this case -- 1)     |
